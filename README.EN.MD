## Tutorial For Create From Basic

- New Project Step 
   
```shell 
$ mkdir project && cd project   
$ npm init project
$ entry point type app.js # this step when init progress
```

- After Init NPM, install nodejs    
```shell
$ npm i express --save
```

- for run your project   
```shell
$ node app.js 
```
- write your application
